Immersive Translate · Design Board · 桃红

三个方向，
一个安静的翻译器。

同一套功能（工具栏图标 · 控制面板 popup · 白名单设置页），三种性格。强调色已统一为桃红，三者的差别便落在质感、版式与图标语义上。每个方向都给出图标的未开启 / 翻译中 / 已开启三态，让你在工具栏一眼看出某站点是否正在被翻译。

挑一个方向 → 我把它落地成真实的 React popup / options + 导出 16·32·48·128 的图标 PNG。

素Quiet

几乎“看不见设计”的克制：纯白、发丝描边、大量留白、唯一一抹桃红。最贴合“隐形”哲学——它安静到像浏览器自带的功能。图标是一条“英→中”的极简双线，下面那条灌满桃红就代表已开启。
