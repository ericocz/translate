海外 $9.99 买断 · 个人开发者两方案落地步骤

2026-06-14 · 面向「无公司主体的个人开发者」 · 已核对 Stripe HK 个人户与 MoR 现状

方案一 · Stripe HK 个人户（护照）

以「Individual」身份注册香港 Stripe，提现到香港收款账户再回流国内
费率 ~3.4% + HK$2.35 上线 数天~数周 有 KYC 冻结风险 税务自理
需要你做的事
  1. 准备护照作为身份证件（HKID 没有时 Stripe HK 接受护照）。
  2. 办香港手机号注册。社区称 +86 也能收码，但 HK 号更稳、更不易被风控。
  3. 开一个香港收款银行账户（提现落地）。主流是 ZA Bank（众安银行）——香港虚拟银行，内地居民凭护照/回乡证可远程开户。⚠ 你说的 Payoneer 未必被 Stripe HK 接受做 payout（HK 通常按 HKD 付到香港银行），需先确认；ZA Bank 更稳妥。
  4. 准备香港网络环境（注册+验证全程需香港 IP）。
  5. 准备地址证明 + 业务描述 + 落地页 URL（含隐私政策，Stripe 会校验）。
  6. 注册 Stripe 时选 Individual，填业务信息、绑定上面的银行账户。
  7. 小额测试提现跑通 Stripe→银行→国内卡全链路，确认到账与汇率损耗。
  8. 接入：你后端已有账号+加密体系，买断激活码自己发；扩展跳转到你域名结账页（Stripe Checkout）。
主要风险：账户与资金可能因「人在内地、无真实香港经营」KYC 错配被冻结。→ 起步别囤余额、及时提现；流水做大后建议转正规主体（香港公司 / 美国 LLC）。

方案二 · MoR 平台（记录商户）

平台是「商户」，你只是供应商；平台代收代缴税、打款给你
费率 ~5% 上线 当天可成 无需主体 / 零 KYC 风险 税务平台代缴
需要你做的事
  1. 选平台：个人最友好 Lemon Squeezy（现属 Stripe）/ Polar；Paddle 审核更严会问业务细节。注：Creem 也是 MoR，你此前蓝图刚去掉它——见下方提醒。
  2. 注册 + 个人实名（护照/身份证），填税表 W-8BEN（非美个人免被预扣美国税，平台引导）。
  3. 绑定收款方式：Payoneer 或 Wise 或银行卡（平台按周期打款，常有最低提现额）。这里 Payoneer 完全适用，是标准选项。
  4. 后台建商品：$9.99 一次性买断；用平台自带的 license key 生成/校验做激活码。
  5. 接入：扩展/后端调平台 Checkout 链接 + Webhook（支付成功回调 → 你后端激活买断）。比自建 Stripe 更省。
  6. 上线，平台自动处理税务 / 发票 / 退款 / 争议。
主要代价：抽成 ~5%（高于 Stripe ~1.6 个点）；对平台有依赖（但风控比 Stripe 个人户温和得多）。

给个人开发者起步的建议：选方案二（MoR）。 方案一省下的费率只有 ~1.6 个点，落到 $9.99 买断上每单仅差 ~$0.16，却要换来「开香港银行户 + 过 KYC + 冻结风险 + 自理海外税」一堆成本与不确定性，不划算。 等月流水做大、确实想省那 1.6 个点时再迁——届时建议直接美国 LLC 拿 ~2.9%，比香港个人户既更合规又更便宜，比 HK 个人户这条「擦边路」干净得多。
⚠ 决策一致性提醒：你的蓝图 V2 此前明确「自有 Stripe 买断、去掉 Creem」，而 Creem 正是一家 MoR。现在方案二等于回到 MoR 路线。建议先想清当时弃用 Creem 的理由（费率？信任？品牌露出？）是否同样适用于 Lemon Squeezy / Paddle——若是，就走方案一/未来美国 LLC；若当时只是嫌 Creem 这一家，换平台即可。
